# Wolfenstein 3D Port

Long-running workspace for experimenting with Wolfenstein 3D ports across multiple languages.

## Layout

- `source/original/` - pristine original id Software Wolfenstein 3D source, tracked as a submodule
- `game-files/` - local proprietary game data from a legally purchased copy; ignored by git

Planned future source folders can live under `source/`, for example:

- `source/modern-c/`
- `source/csharp-sdl3/`
- `source/rust/`
